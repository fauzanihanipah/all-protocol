2. Submenu SSH

m-ssh
================================================================
                MENU  SSH / SSH-WS / SSH-SSL
================================================================
  1) Tambah User SSH
  2) Hapus User SSH
  3) Cek User Login (online)
  4) List User SSH + masa aktif
  5) Buat User Trial (1 hari)
  6) Perpanjang masa aktif user
  7) Hapus user expired (manual)
  8) Restart SSH services
  0) Kembali
----------------------------------------------------------------
